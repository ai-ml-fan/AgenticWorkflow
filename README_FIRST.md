# AI-Powered Agentic Workflow for Project Management

In this repo, you will find all the files to create and run the AI agentic workflow for project management.
## Getting Started

The project is completed in two phases.
1. Phase 1 Code inside the `starter\phase_1` folder in this repo contains the source
for creating different types of agents such as ActionPlanningAgent, AugmentedPromptAgent etc.
These are all defined in the base_agents.py contained in the workflow_agents folder.
The agents are then tested by running the python files of the same names that reside in the phase1 location.
2. Phase 2 Code is located inside the `starter\phase_2` folder. 

## Dependencies

A `requirements.txt` file has been provided in this repo if you want to work on the project locally. Otherwise, the workspace has been configured with all the required libraries. 

#### The python files in the phase_1 location depend on the workflow_agents package. 

#### Phase_2 depends on the workflow_agents package. 

## Project Instructions
You will find instructions for each of the two phases of the project in the README file inside the folder for that phase.

The output file **output_phase1.pdf**
containing the outputs from running all the agents of Phase 1 is provided in the phase_1 folder.

The output file **output_phase2.pdf**
is provided in the phase_2 folder that contains the outputs from running the Agentic workflow to provide the overall development plan for an Email Router.
